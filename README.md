# FPGA Project 1
2025 Fall FPGA

This repo contains the code and data for FPGA Project 1. For more details about the tasks of this project, please refer to the project documentation([old one](docs/project%20description.pdf) and [new one](docs/Homework%204.pdf)).

## Directory Structure
```
fpga_project1/
├── simulator/               # Simulator code (Task 1)
│   ├── test.py              # Test script for the simulator
│   ├── sim.py               # Simulator implementation
│   └── data/                # Test data for the simulator
│
├── PE/                     # PE core implementation
├── requirements.txt         # Python dependencies
├── README.md                # Project documentation
└── docs/                    # Additional project documents
```
